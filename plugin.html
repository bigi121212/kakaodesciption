<플러그인>



사용자에게 특정 화면을 보여주어 값을 선택하게 하고 선택된 값을 이용하여 봇이 질문 또는 답하게
만드는 용도로 쓴다.
파라미터 설정을 통한 플러그인과 봇 응답형식 플러그인으로 구분

[파라미터 설정 플러그인]
되묻기 질문 시 봇이 사용자로부터 받아야할 답안을 편리하고 정확하게 입력할 수 있도록 도와줌
	*중요한 것은 엔티티 설정에서 시스템 엔티티로 들어가 플러그인들을 활성화 해야 사용가능

예) 사용자가 티케팅을 해서 날짜를 봇에게 전달해 주어야 하는 상황

	날짜라는 발화를 사용자로 부터 받는다
	파라미터 만들기 클릭하고 우측 상단에 플러그인을 체크해준다.
	파라미터 명과 플러그인에서 sys.plugin.date 설정
	사용자의 선택값을 받기 위해 '값'창에 "$파라미터명"을 적는다
	되묻기 질문 클릭 후 텍스트상자와 버튼을 만들어준다.
	버튼 만들 때, 범위 지정 또는 직접 지정으로 본인 스스로 설정한다
	봇 응답으로 텍스트형 '티케팅 날짜 @파라미터명 선택했습니다' 저장

	작동)
	사용자:	날짜
	봇:	5번에서 만든 텍스트와 버튼이 뜬다
	사용자: 버튼 클릭 후 날짜 선택 창이 뜨고, 사용자는 날짜를 선택한다.
	봇:	티케팅 날짜 2xxx-xx-xx 선택했습니다

	*나머지 플러그인에 대한 설명은 여기


[봇 응답형식 플러그인]
사용자가 플러스 친구와 관련된 컨텐츠로 연결하는 수단.
	*중요한 것은 이미지 응답으로는 플러그인 불가능하다

예) 친구에게 플러스친구를 공유하고 싶은 상황

	사용사 발화: 공유
	봇 응답형식으로 텍스트형 선택 후, 텍스트 작성
	버튼 추가 후 버튼 설정의 버튼기능에 '플러그인'으로 바꾼다.
	바로 오른쪽 칸을 '공유하기'로 하고 확인 후 저장

	작동)
	사용자: 공유
	봇:	위에 설정한 텍스트 뜨고 공유버튼이 생김
	사용자: 공유 버튼 클릭 -> 친구 선택 창 뜨고 친구를 선택
		->친구 선택 후 친구에게 '플러스 친구 홈'으로 가는 버튼 메세지 전달.

	*나머지 플러그인에 대한 설명은 여기
